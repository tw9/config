
## home directory setting

### shell

http://ohmyz.sh/

Mac
- go `System Perference -> Users & Groups `
- click unlock button
- Right click current User icon
- Enter to Advanced Option 
- Change `Login shell` to `/bin/zsh`


```shell
git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
```
